03Performance Fixes (Deployed Tonight)

Both issues Jake reported are fixed and live.

IssueRoot CauseFixResult
2-3 min response times Sync AI calls blocked the event loop. Sequential round-trips. asyncio.to_thread, parallelized intent classification, trimmed prompt, timeout 90s to 30s. 15-20s
Mobile drops on backgrounding No visibility handler. OS kills fetch when tab backgrounded. Wake Lock API, visibilitychange handler, auto-reconnect, polling fallback. Fixed

04Feature Roadmap

FeatureApproachEffort
Image upload in chat Multipart upload endpoint, Cloudflare R2 storage, signed URLs, inline thumbnails. 1-2 days
CSV report downloads Backend has export patterns. Add /reports/export endpoint, query MongoDB, serve as download. 1-2 days
PDF generation WeasyPrint (HTML/CSS to PDF). Branded templates. /reports/pdf endpoint. ~1 week
Email notifications Triggers in backend, email via Brevo/SendGrid, user preferences. ~1 week
In-app real-time notifications WebSocket per active session. Server pushes events. ~2 weeks

Recommended priority: CSV export and file upload first (live within a week). PDF and email notifications next. Real-time notifications last.

05E-Commerce Vision

Separate storefront at store.trustoffice.app. Stripe Checkout for payments (PCI-compliant, no card data on our servers). Hermes manages inventory, fulfillment, and customer communications autonomously.

Inherits all existing security: TLS, JWT, CORS, profile isolation. No card data stored. ~3-5 week build for MVP with automated operations.

06Monitoring & Reliability

Three new systems went live tonight. We catch issues before users do.

SystemWhat It DoesStatus
Synthetic Canary Hits AI chat endpoint every 30 min. Measures response time. Alerts if >5s (warning) or >10s (critical). Live
Post-Deploy Smoke Test 9 endpoint checks after every deploy: health, auth, CORS, AI providers, bundle hash, polling endpoint. Live
In-App Error Reporting Frontend captures JS crashes. Backend logs all unhandled exceptions. Queryable via admin API. Live
Weekly Audit 5 checks: async safety scan, AI response time, mobile resilience, timeout verification, prompt size. Live
